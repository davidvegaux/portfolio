Ideación

Todo comenzó con una conversación honesta sobre la frustración diaria. Kike Peña y yo sabíamos que había una forma mejor.

Prototipo

Diseñé en Figma los elementos esenciales: sidebar, topnav, widgets modulares. Simple, pero con potencial infinito.

Construcción

Con Cursor AI, transformé el diseño en código funcional. GitHub se convirtió en nuestro punto de distribución instantánea.

Validación

La primera prueba fue mágica: 20 segundos para crear un widget completo. Los stakeholders quedaron sin palabras.

Escalamiento

Integramos el design system completo. Cada output automáticamente cumplía con los estándares visuales de UBITS.
